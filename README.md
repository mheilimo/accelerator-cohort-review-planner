# Accelerator Cohort Review Cadence Planner

A free browser-only programme-operations worksheet for accelerator and
incubator teams. It turns intake, mid-programme mentoring and pre-demo-day
review into one documented cadence across 12 private-company dimensions.

The worksheet is not a startup score, ranking or selection engine. Programme
teams assign the review moment, mentor owner and evidence or change to inspect.
Data stays in the browser and can be copied as an agenda or downloaded as CSV.

## Use it

Open the live worksheet:

<https://mheilimo.github.io/accelerator-cohort-review-planner/>

Or clone the repository and open `index.html` locally. There is no build step,
account, analytics or server-side storage.

## Review cadence

1. **Intake baseline** — document the starting evidence before coaching changes
   the materials.
2. **Mid-programme** — assign mentor attention to changed assumptions and the
   largest unresolved gaps.
3. **Pre-demo-day** — check whether current pitch claims match current operating
   evidence before investor introductions.

The 12 headings mirror DDScore's structured private-company review framework.
The contextual product link is visible in the worksheet; the tool itself does
not score, select, accept or reject a company.
